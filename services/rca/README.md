# Сервис детерминированного анализа RCA

Часть B решения [ADR-0032](../../docs/architecture/adr/0032-логирование-наблюдаемость-и-детерминированный-rca.md),
спецификация [logging-observability-rca.md](../../docs/specs/logging-observability-rca.md).
Реализует конвейер анализа логов и установления первопричины по книге «LLM и
детерминированная инженерия анализа логов и RCA» (Биркин М. С.): факты считает
детерминированный код, языковая модель подключается только на краях.

## Модули (детерминированное ядро, без внешних зависимостей)

Агрегатор `aggregator.py` за один проход O(N) считает восемнадцать блоков фактов
(в том числе временной ряд окна и активность источников во времени) и группирует
строки по сквозному номеру по десяти корреляционным полям. Каталог детекторов
`detectors.py` (D1-D12) потребляет факты и baseline и даёт веса как отношения
правдоподобия; детекторы перерыва в логах, молчания источника, структурного соседа и
демпфера восстановления опираются на временной ряд и граф зависимостей из фактов. Скоринг `scoring.py` переводит сработавшие детекторы в
уверенность байесовским обновлением в шансах с группировкой по максимуму, потолком
0,999, демпфером восстановления, коэффициентом полноты и полосами доверия. Сборка
`verdict.py` даёт вердикт по пятиполевой схеме (статус, уверенность, первопричина,
свидетельства, действие) с реестром свидетельств и гардом «нет цитаты, нет
утверждения». Оркестрация `pipeline.py` (функция `analyze`) связывает всё в один
детерминированный проход.

## Обвязка

Читатель `loki.py` забирает окно логов из Loki (query_range) и разбирает
JSON-конверт канона в записи, сохраняя сырьё строки для дословных цитат. Приложение
`app.py` (FastAPI) отдаёт `GET /health` и `POST /analyze`; последнее принимает окно
логов напрямую или читает его из Loki (с baseline со сдвигом на неделю) и
возвращает факты, детекторы, скоринг и вердикт. Сам сервис логирует по канону
(структурный JSON, service=rca, trace_id).

## Осталось

инженера методом SetFit и оркестратор на краях (языковая модель для разбора запроса
и формулировки отчёта по уже посчитанным фактам).
